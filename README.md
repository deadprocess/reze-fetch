## Requirements
- FreeBSD
- `gcc` — falls nicht vorhanden: `pkg install gcc`

## Build & Run
```sh
sh run.sh
```

## What it shows
- OS Release (`kern.osrelease`)
- Hostname (`kern.hostname`)
